# yt-mood
First ever project!

A CLI tool that recommends YouTube videos based on your mood and how much time you have.

## How it works
- Enter your mood (dark, chill, funny, motivated, sad, bored, learning)
- Enter how much time you have (1-10)
- Get 10 YouTube videos with duration and view counts

## Setup
1. Clone the repo
2. Install dependencies: `pip install requests python-dotenv`
3. Add your YouTube Data API key to a `.env` file: `YOUTUBE_API_KEY=your_key`
4. Run: `python main.py`
